# 🧠 Human Nervous System

A responsive educational website that provides detailed information about the Human Nervous System. The project is designed with an attractive user interface to help users understand the structure and functions of the nervous system.

## 🚀 Features

- Responsive design
- Home page
- Introduction to the Nervous System
- Central Nervous System (CNS)
- Peripheral Nervous System (PNS)
- Brain and Spinal Cord information
- Interactive layout
- Contact section
- Smooth navigation

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

## 📁 Project Structure

```
Human-Nervous-System/
│── index.html
│── style.css
│── script.js
│── images/
```

## 🎯 Purpose

This project was developed as an educational website to provide users with a clear understanding of the Human Nervous System using a modern and responsive web interface.
